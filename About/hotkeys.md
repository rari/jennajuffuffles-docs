# Hotkeys

_Last updated:_ 2026-06-10\
_Applies to Stardew Valley:_ 1.6.15+

Bindings below were captured from **Generic Mod Config Menu** with **Stardew Valley VERY Expanded**, **Aesthetic Valley Fairycore**, **Aesthetic Valley Fairycore Extra Textures**, **Aesthetic Valley Witchcore**, and **Aesthetic Valley Witchcore Extra Textures** installed together. Your keys may differ if you changed them in **Settings > Mod Options** (in-game) or from the title screen (gear icon, lower left).

Mods are grouped by **Nexus collection** and only mods that registered a keybind in GMCM appear here.

***

## Stardew Valley VERY Expanded

* **Automate:** Toggle automation overlay; `U`
* **Better Crafting:** Hold `LeftShift` while opening a crafting menu to use vanilla crafting instead; Modifier `LeftShift` / `RightTrigger`; Favorite recipe `F` (also `ControllerBack`); Search `F3`
* **Better Signs:** Debug; `J`
* **Chests Anywhere:** Toggle chest UI; `B`; Next/previous chest; `Right` / `RightShoulder`, `Left` / `LeftShoulder`; Next/previous category; `Down` / `RightTrigger`, `Up` / `LeftTrigger`; Open search from a chest; `LeftControl` + `F` (also `LeftStick`); Preview hovered chest in search (hold); `Tab` / `LeftTrigger`
* **Convenient Inventory:** Quick stack; `Q` (controller: `LeftStick`); Take all but one from a stack (hold); `LeftControl` + `LeftShift` (controller: `LeftTrigger`); Favorite while selecting (controller); `LeftShoulder`
* **Deluxe Journal:** Toggle overlays; `O`
* **Font Settings:** Open font menu; `LeftAlt` + `F`
* **Generic Mod Config Menu:** Open menu; `Numpad -` (`Subtract`)
* **ItsStardewTime:** Vote to pause (multiplayer); `Pause`
* **Lookup Anything:** Toggle lookup; `F1`; Toggle search; `LeftShift` + `F1`; Scroll; `Up` / `Down`; Page; `PageUp` / `PageDown`
* **NPC Map Locations:** Map settings (while map is open); `Tab`; Toggle minimap; `|` (pipe / `OemPipe`); Cycle tooltip on map; `Space`
* **Social Page Order Redux:** Next sort; `Up` / `DPadRight`; Previous sort; `Down` / `DPadLeft`
* **SpaceCore:** Advanced interaction menu (hold while talking to an NPC); `LeftControl`
* **Sword & Sorcery (Adventure Bar):** Toggle bar; `LeftControl` + `U`; Configure bar; `U`; Bar 1 abilities; `LeftControl` + `1`-`8` (number row); Bar 2 abilities; `LeftShift` + `1`-`8`
* **Trinket Tinker:** Companion interaction; `LeftAlt` + `Right click` (also `LeftStick` + A); Open tinker inventory; `RightAlt` + `.` (period) (also `LeftStick` + X); Next / previous tinker inventory; `PageUp` / `RightShoulder`, `PageDown` / `LeftShoulder`
* **UI Info Suite 2 Alternative:** Calendar; `K`; Quest board; `H`; Special orders board; `J`; Mod options; `F8`; Monster eradication; `F9`; Toggle machine icons; `F10`; Hide trees; `F7`; Animal building tooltip (hold over coop/barn); `LeftControl`; Show all item range; `LeftControl` + `LeftAlt` (hover item)
* **World Navigator:** Open navigator; `N`; More info (cursor) in navigator; `F1`; Describe current guidance; `LeftShift` + `B`

## Aesthetic Valley Fairycore or Witchcore (+ Extras)
* **Fashion Sense:** Hand mirror / quick menu; `G`
* **Furniture Framework:** Place furniture in a slot; `Left click`; Interact with slotted furniture; `Right click`
* **Generic Mod Config Menu:** Open menu; `Numpad -` (`Subtract`)
* **Happy Home Designer:** Give from catalogue (hold while clicking); `LeftShift`; Favorite (hold while clicking); `LeftControl`; Delete held item; `Delete`; Open catalogue anywhere; `L`
* **Precise Furniture:** Move; `Left` / `Right` / `Up` / `Down`; Speed modifier; `LeftAlt`; Lock furniture; `Right click`; Blacklist hovered piece; `Right click` (both use `Right click` in this profile; check **Mod Options** for how the mod distinguishes them)
* **ReMapping Earthy ReSync:** Resync Remapping tokens; `F7`

***

Check Keybinds in Generic Mod Config Menu (upper left) to view all custom keybinds. All keybinds listed here (and more!) can be modified in Generic Mod Config Menu per-mod.
