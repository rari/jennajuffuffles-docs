# JennaJuffuffles Collection Documentation

_Last updated:_ 2026-06-10\
_Applies to Stardew Valley:_ 1.6.15+

Welcome! This documentation provides step-by-step guides for installing, updating, and troubleshooting Stardew Valley mod collections curated by JennaJuffuffles using Vortex. Whether you're new to mod collections or looking to customize your experience, these guides will help you get started and make the most of your collections.

This documentation covers multiple collections including Stardew Valley VERY Expanded and the Aesthetic Valley collections. Use the navigation below to find what you need, or explore the [About the Collections](About/index.md) page to learn more about each collection.

{% hint style="warning" %}
**Playing with mods in multiplayer?** It is strongly recommended that all players have matching mod loadouts: the same collections, mods, versions, and configurations. Mismatched setups can cause crashes, desyncs, or connection failures. See [Multiplayer Support](About/index.md#multiplayer-support) for best practices.
{% endhint %}

### Installing Collections

**New to mod collections?** Start with these basics. They match the steps on each [Nexus collection page](https://www.nexusmods.com/games/stardewvalley/collections/tckf0m). For verifying game files, SMAPI setup, optional mods, and store launcher configuration, see the full guide linked below.

1. **Install [Vortex](https://www.nexusmods.com/site/mods/1)** (required)
2. **Manage Stardew Valley:** In Vortex, go to **Games** and click **Manage** on Stardew Valley. If you already use Vortex for Stardew, click **Purge** on the Mods panel first.
3. **Open your Mods folder:** On the **Mods** panel, click **Open** and select **Game Mods folder** (the third option).
4. **Clear leftover files:** Remove any unmanaged files or folders from that Mods folder.
5. **Add the collection:** On the collection's Nexus page, click **Add Collection**, then follow the Vortex prompts.

   **New profile (start here)**

   * [Stardew Valley VERY Expanded](https://www.nexusmods.com/games/stardewvalley/collections/tckf0m): Create a **new profile**. This is the default.

   **Add to your SVVE profile**

   * [Oops! All Portraits: Nyapu](https://www.nexusmods.com/games/stardewvalley/collections/falitk)
   * [Aesthetic Valley Fairycore](https://next.nexusmods.com/stardewvalley/collections/tjvl0j) **or** [Aesthetic Valley Witchcore](https://next.nexusmods.com/stardewvalley/collections/g14kxi): Pick **one**. They are mutually exclusive as of May 2026. You can also choose **Create new profile** from the first pop-up.

   **Add to any working profile**

   * [Controller Support](https://www.nexusmods.com/games/stardewvalley/collections/crx9cn)
   * [Aesthetic Valley Fairycore Extra Textures](https://www.nexusmods.com/games/stardewvalley/collections/i8ulwo/)
   * [Aesthetic Valley Witchcore Extra Textures](https://www.nexusmods.com/games/stardewvalley/collections/ufo0fl/)

**Installation paths:**

- **Nexus Premium:** One-click installation through Vortex
- **Free users:** Guided installation through Vortex (manual download prompts per mod)
- **Manual installation:** Possible but not recommended; see [Manual Installation](Installation/manual.md)

{% hint style="info" %}
All mods are pre-configured to work together. For the complete walkthrough (game verification, optional mods, SMAPI, and connecting your store launcher), continue to the full install guide.
{% endhint %}
