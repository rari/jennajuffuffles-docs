# Vortex

*Last updated:* 2025-11-25  
*Applies to Stardew Valley:* 1.6.15+

---

Vortex is the recommended mod manager for most users, especially Windows users. It provides robust collection support, automatic configuration management, and easy updates.

**Key Benefits:**
- One-click or guided collection installation
- Automatic configuration file management
- Easy collection updates
- Profile system for managing multiple setups
- Built-in conflict resolution tools
